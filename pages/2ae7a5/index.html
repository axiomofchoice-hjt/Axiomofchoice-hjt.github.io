ACM-XCPC 回忆录

# 我的 OI 生涯

我是小学五年级开始接触 OI 的。那时候可是连函数都没学到就去参加比赛的猛男，我记得有一题要用到递归，类似全排列那种题，我直接写了 (1 + 2 + 3 + 4 + 5 + 6 + 7) 个 for，竟然还拿了满分。

OI 确实让我喜欢上了编程。我可能算是一个“完美主义”者，而这个世界上有两个事情可以满足我完美主义的强迫症，一个是 MC，一个是编程。这两者都是那种 我可以做很多事情，而且总是有方法控制所有的细节 的游戏。编程也算一种游戏吧毕竟都能刺激多巴胺（不是）。

那时候学的还是 Pascal，初中老师和同学一直让我换 C++，其实我初一就学了一些 C/C++ 的基础。甚至我 NOIP 初赛都是 C++ 语言参赛的。但是因为我还在用 Pascal 的图形库写游戏，直到高二的时候遇到了 EasyX（古老的图形库）（但是比 Free Pascal 自带的库强）我才彻底放弃 Pascal。

游戏是不可能开源的，我写游戏是为了自娱自乐，别人也不可能喜欢我的游戏。

哦，其实别看我这么早接触 OI，实际上，初中、高中的学校没有多少 OI 氛围，老师也不管我，所以这 6 年我的 OI 水平几乎没什么变化。我高一才听说了线段树这种东西。要说为什么我这么不争气，那可能是写游戏太爽了吧，也可能是每年的二等奖让我产生了，没拿一等奖是我运气不好而不是实力不够的想法。

希望后生们不要犯我这位菜鸡才会犯的错。

我高中三年 NOIP 都拿的是二等奖，但是好像并没有什么用。每个学校自主招生的门槛都是省一起步。高三那年，NOIP 结束后，我也成为了退役狗。OI 带给我很多知识和思考。没有 OI，也许我只会沉迷玩游戏而不会沉迷写游戏。（确实，没学 OI 的时间都花在写游戏里了）

我以为我可能就这么退役了，直到得知高考分数，发现 985/211 学校没有希望了。因为 HDU 的 ACM 还是有点名气的，于是我就来了 HDU，成为了一个 ACMer。不过我觉得这也没什么不好的。

# ACM 的开始

大一是我 ACM 水平增长最快的一年。因为不知道我和学长学姐们的差距，我拼命学新算法，打 CF，有空就看 OI-Wiki。另外我没有组队，没有拿过牌，这些都是让我默默奋斗的原因。（当然现在也不是说没有动力了，只是我不再和学长做比较了，而是队和队做比较）我意识到知识的广度可能是一个重要的瓶颈，所以我的主要战略是尽可能地扩充我的模板库（现在导出到 pdf 有 192 页了），知识的深度主要靠智商和队友来凑。

说实话，大一之前，我只有 NOIP 普及+/提高的实力，大一之后，我可能就是区域赛金牌垫底的实力了。不过我一直没有记录生活的手段，不知道以前的自己的确切实力。

大学的一开始，同届的就有 4 个进队了，其中三个特别猛。我只是个萌新，为了防止孤立，我拉了三个猛男中的凯爷聊天（doge）。凯爷也是我现在的队友。那时我还不懂规矩，就老是在 ACM 新生群里问我不会的题，有时就问得有点难或者冷门（其实我不知道有多难或冷门），没得到答案。emmmmm。后来我就逐渐有了不依赖别人的习惯。

好喜欢 OI-Wiki 啊，虽然它对现在的我已经没什么用了（差不多看完了），但是它真的可以指明我学习的方向。OI-Wiki 的高级一些的算法不算多，我挺想为它补充但是怕我写的博客太烂了。

大一，我们的教练——老刘指定的入队规则是，CF连续三次蓝名 (1600+) 或者参加选拔赛。我这才明白 CF 的重要性，为了早点进队，熬夜肝 CF，有一场是到 2:30，早上 8:00 的早课依然去上。当年的热情啊，现在天天咕 CF 的我自愧不如。

打 CF 的时候为了激励自己，我开始坚持写题解，然后发给别人看，这样就可以强迫别人监督自己。

再往后两个月，我第一次紫名，不过不稳定，回过 3 次蓝色。有一次不知道规则，edu 场倒着开题，而且前两题居然来不及做，猛掉 152 分回蓝。再下一场遇到 div1 2 分场，是一个手速场，我在 div2 以惊人的手速打到 22 名，猛涨 167 分回紫。所以有紫名水平的童鞋回蓝了不要害怕，只要抓住 div1 2 分场的机会说不定反弹更高 (doge)。

紫名的时候 div2 还是算分的，到了橙名后，div2 随便打，div1 又容易掉分，所以天天咕，题解也不怎么写了。

打了这么久的 ACM，我也知道我某种意义上到达了瓶颈。HDU 一队的三个学长，实力、训练时间和训练强度都远远超过我。在我眼中，一流水平 = 清华北大，二流 = HDU 一队，三流 = Null，四流 = 我。我作为四流能理解三流的操作，可以将他们当成努力的目标，但是无法理解更强的人。

# 2020 暑假

再后来，大一快结束的时候，出现了巨大的转变：老刘要我们组队了，而且是自由组队，我们自己协商组队。老刘发这个消息的时候，我还在打游戏，打完后才意识到问题的严重性因为大爹和猛男都抱团了。当时我和凯爷没人要，凑在一起还缺一个，我又太过腼腆，幸好白老师找我组队。

于是一个新的队伍诞生了，队名叫吾有一數名之曰誒。

凯爷那天说他要好好学 dp。所以从这天开始我 dp 水平就再也没进步了，dp 题推给凯爷多好 (doge)。像 dp 这种状态复杂的算法，debug 多困难，所以我 dp 一直不行。白老师负责了字符串，对我来说和 dp 一样都是状态复杂的算法。

我也没什么很强的地方，除了冷门科技，这我还是很有信心的。队里我主要承包了计算几何。另外我代码能力惊人，还负责了很多题的代码手。

大一结束的暑假，就开始留校打牛客和 HDOJ 总共 20 场多校训练。我一直觉得我们队各方面都不算突出，还有两个刚组队的大二萌新，和大三的大爹怎么比。没想到多校表现还不错，排名第二。一队遥遥领先，yyds，肯定比不过。虽然第二，但是和第三第四差别不是很大，经常能遇到被他们吊打的比赛。

多校总结了很多经验。我感觉我们队的手速相对还是挺稳定的，签到题不敢说能做很快，但是很少卡住，中等题也可以快速切掉。一旦到了金牌题，变化就非常丰富了，要不就是别人都会做我队连思路都没有，要不就是我队随手切掉，结果一看全国只有 2 个队做出来。

之前我还想学一学模拟退火的来着，结果多校别人退得出，我退不出。现在我队没有模拟退火选手。

多校后我队意识到缺个多项式选手，我只好去死磕多项式，学了一个礼拜，多项式全家桶到生成函数，各种卷积姿势。奈何天赋太差做啥啥不会，一个礼拜后就颓废了。

我还是我队唯一 Java 选手，只不过从没用过，因为能用 Python 就用 Python 了。

# 2020 CCPC 秦皇岛

大二上这一学期很忙，有很多比赛。很快就迎来了我的第一场区域赛，CCPC 秦皇岛。我队上来就开始传统艺能——签到，但是后来真正要表现技术的时候却一题没过。不过得到老刘赏识（雾）得到了 CCPC 长春入场券了。

# 2020 CCPC 长春

CCPC长春，我没去知乎写比赛感想，细节忘了。首先是一道签到的背包题，白老师以为贪心可过，wa 了一发。后来我才叉掉贪心，赶紧背包，出师不利啊。凯爷当时要做一道构造，天坑啊，出不了。然后就是一道挺麻烦的树上启发式合并，我那时没板子，只知道这算法的流程，用白老师板子写的。这题写得最久，写完只剩两个小时左右了。

再然后是一道 dp，我口胡了思路想让 dp 大师凯爷做可是凯爷不自信，我只好开莽，一通操作竟然 1a 了。然后是一道启发式合并套数论，启发式合并是白老师的想法，数论是我设计的，然后作为首席代码手我又上了，我又 1a 了。那天手感有点好啊。最后惊险拿金，最后两题但凡有点失误都会变成银了。

不过榜歪得离谱，最复杂的树上启发式合并过的人比 dp 和数论多。我们学校其他队伍感觉都被歪榜影响了，没去开其他题，挺惨的。

CCPC 结束，我们又得到老刘赏识（啊这），拿到了 CCPC final 的入场券。感谢老刘的认可！

# 2020 ICPC 南京

比赛前老刘找到我们队说你们要勇于开题，不要用保守策略。然而我们队最擅长保守了（划掉）。虽然南京是我们队打得最好的一场，但是比赛过程可能和老刘想的不那么一样吧。

当时签到题做得一般。中场的时候，两位队友过了 dp（好像是吧），我过了吉老师线段树，这题虽然板子但是吉老师线段树可能没完全普及吧。这奠定了我们超高排名的基础，那时候是第 4，一直到封榜。

然后能做的题只有 A 和 D，A 是构造，我可能没睡好觉脑子不清晰，明明已经想到两个结论 R1 和 R2，R1 推导出 R2，结果我放弃了 R1 自我误导得出了假的思路（这思路与 R1 矛盾）。事实上差点就可以用 R1 + R2 推导出 R3 从而得到相反的正确思路。我就这么错失了 AC 的机会。

D 也一样，我虽然没仔细想，但别的队说随机可过（非正解）。这说明 A D 两题都不是传统意义上的出线题。

比赛结束，第 4 掉到第 6，失去了出线机会。不过做人要知足 (doge)。

# 大二上总结

什么省赛，天梯，其实含金量没区域赛高就懒得讲了。打得也不是挺好。另外打得不好的比赛是 ICPC 小米邀请赛。当时我已经预感到我们要凉，因为是杜老师和吉老师出题，在多校里只要是他们出题我们都会崩。不过这次旅行还比较舒适，虽然打铁，但是零食吃了很多啊，而且见到了邓神邓学长。

我感觉 ACM 学到后来，题目慢慢向毒瘤方向发展了。比如计算几何，很多操作都比较小众，没有什么题解，写起来又 WA 不断，导致很多遗留问题无法解决。

写回忆录的时候已经是大二上的期末。

# 大二下

好久没来写回忆录了。大二下的课是真的多，所有比赛刚结束，我疯狂补作业 + 复习到期末考，勉强算是熬过来了。

首先是 ICPC EC-Final，遗憾拿银。被一道简单题卡了，原因可能是算法里有浮点三分。

然后打了 ICPC 银川，金牌但是很一般的成绩。

银川的下一礼拜是上海华为训练营，捞了好多赠品（）。

再下一礼拜是 CCPC-Final，体会到了被暴打的感觉，拿了铜。我们队最后一战竟然是这种方式结束的。

至此，我大二的所有比赛完结。

# 2021 暑假

在组队的时候，我还不知道出线的事情。我留下来打 2020 赛季的主要原因是，感觉 4 块金牌有点少，想拿多点牌子。

暑假开始，又到了一年一次组队环节。事实上组队什么的也毫无悬念，学弟最强队已经定下了，只好在同届里往强的挑（doge）。而“A数”队只离开了白老师，凯爷和我决定把汤神选为新的 C 位（doge），队名也从“A数”变成了“B数”。

然后恭喜我自己，ICPC World Final 扩招，让南京校第 5 的“A数”队出线了，难以置信。

知道出线的消息后，我发现我的境况有点尴尬。要想出第二次线，不是没有可能，只是没什么用（因为已经是 WF 选手了）。所以退役了多好。但是队伍都组好了，抛下队友也不可取。不过我的目标本来也不高远，只要捞点牌子就好了，训练很大程度只是为了维持手感了。换句话说，我退役一半了。

不过大二就退役也不是大多数人的操作，我还是把自己定位为大多数人吧。

多校打下来，整体感觉和以前差不多。具体也不知道实力咋样，只知道比学弟队弱。我队相较于学弟队，虽然有赢有输，但是学弟队状态好起来可以杀疯，我们打得好的基本赢在罚时。终究甘拜下风。（说起来南京也是赢在罚时来着）

# 大三上

大三上基本打完所有区域赛了。

前三场（CCPC 广州，ICPC 济南，CCPC 哈尔滨）问题不是很大，至少都保住了金牌。但是 ICPC 南京暴露了一些问题致使我队只拿了银牌。南京中后期开了两题，一题是我写的数论题（对因子搜索），一题是两位队友写的数据结构题。可惜我对有复杂状态的代码的调试太不熟练（算是我的一大弱点了），更可惜的是我赛后 8 分钟调试过样例。而两位队友的题也是赛后几分钟才完全想明白。可以说是丢失了所有的金牌题，银牌也毫不奇怪了。

南京后我也有点受打击，精神比较恍惚。大概不甘心以这种遗憾的方式退役吧。（虽然后来还有几场比赛）

学弟队是真的 nb，两次出线 wf，相比之下，我队不光没出线，南京甚至只拿了银牌。上个赛季还是学长的五点共圆队统治杭电，这个赛季可以说是学弟的杀戮降临队统治了杭电（我还是太平庸了）。

一切都差不多结束了，我也该找实习了。
