# 1. 简介

一个解密游戏，规则由玩家发掘，难度较高。

# 2. 攻略 1-10

# 2.1. 第 1 关

规则

  • 在 555 次机会里猜出正确数字。
  • 会提示正确数字与猜的数字的大小关系。

攻略

每次点击答案的范围内最中间的数字，并缩小答案的范围。

比如一开始答案范围是 [1,31][1,31][1,31]，第一次点击 1+312=16\tfrac{1+31}{2}=16​2​​1+31​​=16。如果这个方块是蓝色，答案范围缩小为 [1,15][1,15][1,15]，第二次点击 1+152=8\tfrac{1+15}{2}=8​2​​1+15​​=8；如果这个方块是黄色，答案范围缩小为 [17,31][17,31][17,31]，第二次点击 17+312=24\tfrac{17+31}{2}=24​2​​17+31​​=24。

原理

二分查找算法。

作为新手引导关卡当然要选择广为人知的算法了。
