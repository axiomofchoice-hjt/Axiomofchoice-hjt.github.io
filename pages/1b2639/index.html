# 2. 费马 - 欧拉素数定理

我们先看一个更简单的定理：形如 (4n+3)(4n+3)(4n+3) 的素数不能表示为一个范数

证明：设该素数为 ppp，假设 a2+b2=pa^2+b^2=pa​2​​+b​2​​=p 是成立的
