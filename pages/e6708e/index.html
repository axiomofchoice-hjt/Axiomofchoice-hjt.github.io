Github 仓库地址 (opens new window)

大家好！我是 HDU-ACM 现役队员 Axiomofchoice。这里是新模板的导航页，并且正在施工中。

另外，我对一些偏冷门的算法（题目少，题解少）的代码非常不自信。我认为这个板子最大的作用应该是让读者知道这个算法的存在，找到学习的方向。

如果在这里你找到了你想要的东西，别忘了将它拷贝到自己的板子上，转化为自己的能力。

感谢大家支持！（由于板子放该网站会出现问题，只能用链接跳转到博客园）

数学 (opens new window)：关于数学的一些板子，比如 CRT、线性筛、多项式全家桶、高斯消元、高精度。

图论 (opens new window)：关于图论的一些板子，比如最短路径、LCA、二分图匹配、网络流。

计算几何 (opens new window)：关于计算几何的一些板子，比如凸包、旋转卡壳、半面交、Delaunay 三角剖分。

数据结构 (opens new window)：关于数据结构的一些板子，比如线段树、LCT、莫队。

其他 (opens new window)：包含了搜索、动态规划、字符串算法，也包含了编程技巧、语言技巧、卡常等。

结论 (opens new window)：关于一切的结论，比如莫比乌斯反演、组合数公式、矩阵树定理。
